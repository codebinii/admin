# Sesión 2026-03-09

## Funcionalidades Implementadas

### Módulo Hash — Generación de Claves (`GET /api/hash/clave`)
- Nuevo endpoint integrado al módulo Hash existente
- Parámetros opcionales: `longitud` (default 12), `numeros`, `minusculas`, `mayusculas`, `especiales` (todos true por defecto)
- Generación con `random_int()` (CSPRNG — criptográficamente seguro)
- Validación de charset: al menos uno activo, error 422 si todos son false
- Fix posterior: regla `boolean` de Laravel no acepta strings "true"/"false" en query params — cambiado a `in:0,1,true,false`

### Módulo DV DIAN (`POST /api/dv`)
- Calculadora del Dígito de Verificación según algoritmo oficial DIAN Colombia
- Recibe array de NITs (1 a 100 por request)
- Acepta formato con o sin guión (`900123456` o `900123456-7`)
- Errores por NIT individual no interrumpen el batch completo
- Algoritmo: pesos [3,7,13,17,19,23,29,37,41,43,47,53,59,67,71] de derecha a izquierda, módulo 11
